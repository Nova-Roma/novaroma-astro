Join a hands-on session on choosing and pairing typefaces for maximum impact.
We will analyse Swiss classics and deconstruct how typographers balance
character with neutrality. The session closes with a collaborative exercise
where attendees refine interface typography for legibility and tone.

## Key Takeaways

- Type anatomy basics
- Pairing fonts for contrast
- Readability in UI
- Building modular type scales for product design
